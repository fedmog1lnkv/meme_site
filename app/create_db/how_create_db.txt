DEGUF = True - режим дебагинга

Перед тем как запустить веб-приложение необходимо создать базу данных. Это можно сделать с помощью функции, находящейся в модуле "create_db". В данном модуле есть файл "sq_db.sql"
В нём прописаны все таблицы и их колонки.

Для запуска веб-сервиса необходимо выполнить файл "main.py"
